# FastAPI - ApiDoc
Api Documentation FastAPI Home Work


For run this hw ->

1 - create venv 
``` text
    python -n venv .venv 
```
or 
``` text
  python3 -m venv .venv
```
2 - activate 
---2.1 - Windows 
  ``` text
  .\.venv\Scripts\activate
```
---2.2 - linux \ macos
   ``` text
  source .venv\bin\activate
```
3 - installing libs
  ``` text
  pip install fastapi, uvicorn
```

4 - run in run.py
5 - go to 
``` text
  http://127.0.0.1:8000/docs
```
its fastapi Swagger UI
